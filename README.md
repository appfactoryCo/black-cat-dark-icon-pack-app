# Black Cat Dark Icon Pack & Wallpapers

Minimal, sleek, and stealthy **AMOLED icon pack** with matching wallpapers.  
Designed for a clean premium dark home screen aesithetics.

---

## ✨ Features

- Handcrafted dark icons  
- AMOLED-friendly deep black design  
- Minimal and consistent style  
- High-quality matching wallpapers  
- Regular updates  
- Icon request support  

---

## 📱 Supported Launchers

This icon pack works with most custom launchers, including:

- Nova Launcher  
- Niagara Launcher  
- Lawnchair  
- Smart Launcher  
- Hyperion Launcher  

> Stock launchers may require a third-party launcher.
